Install
-----------------
A selection of gems or applications updated during past week.

* Sassaparilla
  css3 - 
  http://sass.fffunction.co/
  A css framework on compass focused on responsive designs.

* Tern
  js - 
  http://ternjs.net/
  Stand-alone code-analysis engine for JavaScript, with plugins for vim, emacs, ST.

* NProgress.js
  js - 
  http://ricostacruz.com/nprogress/
  A nanoscopic progress bar. Featuring realistic trickle animations to convince your users that something is happening.

* Steroids
  mobile - 
  http://www.appgyver.com/steroids
  On the top of phonegap, but guilding thing in an hybrid way, like semi-native and faster.

* Brick
  web - 
  http://mozilla.github.io/brick/
  UI component for the web by Mozilla.

* Apropos
  css3 - 
  https://github.com/square/apropos
  A compass extension for handling responsive image sizes.

* IEvms
  tool - 
  https://github.com/xdissent/ievms
  Debugging js/css for IE is hell, this can help by setting up virtual machines for each version.
